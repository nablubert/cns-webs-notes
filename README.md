# CNS Web Notes

一个用于归档和发布多个独立 HTML 页面的仓库。本仓库的定位是存放各类静态网页素材与笔记，不针对任何特定域名或网站。

## 项目简介

本仓库主要用于收集、整理和发布独立的 HTML 页面。这些页面可能涉及技术演示、学习笔记、文档片段或实验性项目等。所有页面均为独立文件，不依赖复杂后端环境，可直接通过浏览器打开查看。

## 目录说明

```
.
├── README.md          # 本文件
├── pages/             # 存放独立 HTML 页面的主目录
│   ├── example1.html  # 示例页面
│   └── ...
├── assets/            # 页面引用的公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── docs/              # 额外的文档或笔记（可选）
```

- `pages/`：所有独立 HTML 页面存放于此。每个页面应有清晰的命名，并尽量保持自包含。
- `assets/`：页面引用的公共资源文件。建议使用相对路径引用。
- `docs/`：存放与页面相关的说明文档、笔记或参考资料。

## 页面归档说明

- 本仓库中的页面均为独立归档，每个页面可以单独使用或分享。
- 页面之间没有强关联性，你可以按需浏览或下载单个页面。
- 页面内容可能包含实验性功能或未完成的笔记，请注意甄别。
- 部分页面可能依赖外部资源（如 CDN），需联网才能完整显示。
